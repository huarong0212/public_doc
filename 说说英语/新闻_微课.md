## 新闻列表
- 接口名称: news/search
- 请求参数
```
{
    user_id: 用户ID
    type_id: 分类ID(1:微课，2:知识点，3:今日热点，4:英语练习题，5:小升初试卷，6:初升高试卷，7:音频微课，8:视频微课，9:词汇，10:语法，11:句型，12:作文，13:听力)
    period: 0小学，1中学
    flag：0所有，1推荐
    page: 页码
    page_size: 每页数量
}
```
- 返回参数
```
{
    code: 0|1
    msg: 消息
    data: [
        {
            id
            title: 标题
            img: 图片
            url: 视频或音频
            keywords: 关键词，多个用逗号隔开
            type_id: 所属分类ID
            type_name: 分类名称
            period: 0: 小学，1: 初中
            flag: 推荐: 1，热门: 2
            author: 作者
            pv_num: PV数量
            ip_num: IP数量
            user_num: 学习人数
            unit_num: 系列单元数量
            sort: 排序字段
            add_time: 添加时间
            add_date: 添加日期（20170904）
            
            is_goods: 是否收费 0:不收费; 1:收费
            is_charged: 是否已购买 0: 未购买; 1:已购买
            charge_num: 已购买人次
        }
    ]
}
```

## 回复

```
{

    news_id: 新闻ID
    follow_id: 0
    content: 回复内容
    notice: {} //
}
```
- 返回参数
```
{
    code: 0|1
    msg: 消息
    data: {
        id:

        news_id: 新闻ID
        follow_id: 回复ID
```
