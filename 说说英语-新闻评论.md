# 接口列表

## 1. 首页
- 接口名: News/index
- 请求参数
```
{
    period: 年级 0：小学，1：初中
}
```
- 返回参数
```
{
    
    code: 0|1
    msg: 消息
    data: {
        redian:[
            {
                id
                title
                img
                keywords
                type_id
                period
                flag
                author
                pv_num
                ip_num
                sort
                add_time
                add_date
            },
            ...
        ],
        cihui:[
            同上
        ],
        yufa:[
            同上
        ],
        juxing:[
            同上
        ],
        zuowen:[
            同上
        ],
        tingli:[
            同上
        ],
        weike:[
            同上
        ],
        shequ:[
            {
                id: 话题ID
                user_id: 发表人
                user_name: 昵称
                face: 用户头像
                content: 话题内容
                images: [
                    'http://xxx',
                    'http://xxx'
                ],
                notice: {}
                type: 话题类型
                flag: 话题标签
                sort: 排序
                follow_count: 回复数量
                agree_count: 点赞数量
                status: 话题状态
                add_time: 添加时间
                agreed: 已点赞 ? 1 : 0
            }
            ...
        ],
    }
}
```

## 2. 回复
- 接口名: News/follow
- 请求参数
```
{
    user_id: 回复人
    news_id: 新闻ID
    follow_id: 0
    content: 回复内容
    notice: {} //
}
```
- 返回参数
```
{
    code: 0|1
    msg: 消息
    data: {
        id: 
        user_id: 回复人
        news_id: 新闻ID
        follow_id: 回复ID
        notice: {}
        status: 状态
        add_time: 回复时间
    }
}
```

## 3. 新闻回复列表
- 接口名: note/followlist
- 请求参数
```
{
    news_id: 新闻ID
    page: 页码
    limit: 每页数量
}
```
- 返回参数
```
{
    
    code: 0|1
    msg: 消息
    data: {
        page_count: 总页数
        list:[
            {
                id: 
                user_id: 回复人
                user_name: 昵称
                face: 用户头像
                news_id: 新闻ID
                content: 回复内容
                follow_id: 0
                notice: {}
                status: 状态
                add_time: 回复时间
            }
            ...
        ]
    }
}
```
