# 接口列表

## 1. 回复
- 接口名: News/follow
- 请求参数
```
{
    user_id: 回复人
    news_id: 新闻ID
    follow_id: 0
    content: 回复内容
    notice: {} //
}
```
- 返回参数
```
{
    code: 0|1
    msg: 消息
    data: {
        id: 
        user_id: 回复人
        news_id: 新闻ID
        follow_id: 回复ID
        notice: {}
        status: 状态
        add_time: 回复时间
    }
}
```

## 2. 新闻回复列表
- 接口名: note/followlist
- 请求参数
```
{
    news_id: 新闻ID
    page: 页码
    limit: 每页数量
}
```
- 返回参数
```
{
    
    code: 0|1
    msg: 消息
    data: {
        page_count: 总页数
        list:[
            {
                id: 
                user_id: 回复人
                user_name: 昵称
                face: 用户头像
                news_id: 新闻ID
                content: 回复内容
                follow_id: 0
                notice: {}
                status: 状态
                add_time: 回复时间
            }
            ...
        ]
    }
}
```
